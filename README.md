### Presentación del proyecto 
- FarmaExpres:
    - Una plataforma de gestión empresarial diseñada para una pequeña farmacéutica.<br>
     Su propósito principal es administrar y controlar el inventario de medicamentos y productos farmacéuticos de manera diferenciada por sede, operando bajo un sistema centralizado.<br>
     El sistema permite acceso concurrente de múltiples usuarios, garantizando:
      - Integridad de datos  
      - Consistencia  
      - Disponibilidad en tiempo real
